# 직무역량 평가 대시보드

## 아키텍처

```
이력서 PDF (사람인)
    ↓ pdfplumber 추출
키워드 스코어링 (AI)
    ↓
6개 직무 섹터별 KPI 점수
    ↓
Dashboard.jsx (레이더 차트 + 섹터 카드)
```

## 6개 직무 섹터

| 섹터 | KPI 예시 |
|---|---|
| 생산 | 설비 가동률, 불량률, 생산성 |
| 장비 | 예방정비율, MTTR, 장비수명 |
| 품질 | 불량률, 고객 클레임, 공정능력 |
| 연구개발 | 특허 건수, 논문 실적, 개발 일정 |
| 인사 | 이직률, 채용 소요일, 교육 이수율 |
| 재무회계 | 정확도, 마감 준수율, 감사 지적사항 |

## 핵심 파일

| 파일 | 역할 |
|---|---|
| `frontend/src/pages/Dashboard.jsx` | 메인 대시보드 UI |
| `backend/modules/recruiting/pipeline.py` | 이력서 스코어 생성 |
