# 사람인 (Saramin)

## 개요

한국 주요 채용 포털. AEON 파이프라인이 자동으로 로그인·이력서 수집에 활용.

## 연동 정보

| 항목 | 내용 |
|---|---|
| URL | hiring.saramin.co.kr (기업 채용 관리) |
| 로그인 방식 | ID/PW → Selenium 자동화 |
| 인증번호 SMS | 경우에 따라 UI에서 직접 입력 필요 |
| 계정 저장 위치 | `backend/auth/users.json` (사용자별 분리) |

## AEON 파이프라인에서의 역할

1. JWT로 사용자 식별
2. users.json에서 해당 유저의 saramin_id/pw 조회
3. subprocess로 pipeline.py 실행 → 사람인 자동 로그인
4. PDF 이력서 수집 → AI 스코어링
