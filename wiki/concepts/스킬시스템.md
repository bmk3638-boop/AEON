# Claude 스킬 시스템

## 개념

스킬 = Claude가 특정 분야에서 판단·출력하는 방식을 바꾸는 학습된 능력.
`/skill-name` 으로 호출하거나, 트리거 조건이 충족되면 자동 실행.

## 설치 위치

```
~/.agents/skills/     ← 전역 (모든 프로젝트에서 사용 가능)
프로젝트/.agents/     ← 프로젝트 전용
~/.claude/skills/     ← Claude Code 설치 경로
```

## 사용법

스킬은 **자동 실행되지 않는다**. 명시적 호출 필요:
```
/wiki          → 볼트 구조 관리
/save          → 현재 대화를 볼트에 저장
/autoresearch  → 주제 자동 리서치
/wiki-ingest   → 외부 자료 볼트에 추가
/wiki-lint     → 볼트 상태 점검
/wiki-query    → 볼트 내 질문
```

> [!key-insight] 핵심
> 스킬이 낮아 보이는 이유는 설치 부족이 아니라 호출 부족이다. 23개 모두 준비된 상태.
