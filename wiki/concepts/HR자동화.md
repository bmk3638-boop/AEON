# HR 자동화

## 핵심 정의

반복적인 HR 업무(채용·온보딩·평가·퇴직)를 AI와 RPA로 대체하여 HR 담당자가 전략적 업무에 집중하게 하는 것.

## AEON의 자동화 범위

| 단계 | 자동화 내용 | 현재 구현 |
|---|---|---|
| 채용 | 이력서 수집·스코어링 | ✅ (사람인 파이프라인) |
| 채용 | 직무역량 대시보드 | ✅ (6개 섹터) |
| 온보딩 | 서류 자동 검증 (고용보험+원천징수) | 🔄 (ocr-review-skill) |
| 평가 | KPI 산출 | ⏳ |
| 퇴직 | 퇴직 처리 자동화 | ⏳ |

## 자동화 설계 원칙

1. **설계 → 구축 → 운영이관** 전 과정 책임
2. 고객 컴플레인 제로 수준의 완성도
3. 다중 사용자 격리 (회사별 계정 분리)

## 비즈니스 임팩트

- 채용 담당자 이력서 검토 시간 80% 단축 (예상)
- 오탈자·중복 처리 오류 제거
- 데이터 기반 채용 의사결정 가능
