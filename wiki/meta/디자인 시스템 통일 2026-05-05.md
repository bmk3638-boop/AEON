# 디자인 시스템 통일 2026-05-05

## 결정

`hr-tools-web` 실행본은 `C:\Users\qoaud\Projects\hr-tools-web`가 기준이다. 2026-05-05 작업에서 검은색 중심 화면과 페이지별 다른 색감을 걷어내고, AEON 자체 기술처럼 보이도록 외부 컨설팅사명을 웹사이트와 API 응답명에서 제거했다.

새 색상 비율은 다음처럼 잡는다.

| 비율 | 역할 | 색 |
|---|---|---|
| 60 | 기본 배경 | `#F4F7F2` |
| 30 | 정보면·보조면 | `#FFFFFF`, `#E7EFE7`, `#263B36` |
| 10 | 행동 버튼·강조 | `#2B8C7E`, `#9DDBCF` |

## 반영 내용

- 홈은 HRM 6영역을 15살도 이해할 문장으로 설명한다: 뽑기, 키우기, 평가하기, 보상하기, 오래 일하게 돕기, 퇴사 정리.
- 외부 회사명은 화면에서 제거하고 `AEON 역량지도`, `AEON 업무분류`, `AEON 몰입신호`로 제품 언어를 통일했다.
- 개발관리와 유지관리 화면은 같은 카드 반경, 같은 배경색, 같은 청록 CTA를 사용한다.
- 평가 템플릿은 `AEON 역량지도 프레임워크`로 보이게 바꾸고, 백엔드 source도 `aeon_competency_map`으로 바꿨다.
- 채용 자동화 화면의 검은 로그 터미널은 사용자용 진행 상태 카드로 대체했다. 서버 로그와 디버그 문장은 화면에 직접 노출하지 않는 방향이다.
- 로그인, 회원가입, 챗봇, 대시보드, 성과관리, 지원자, 면접, OCR, 약관 페이지의 기본 색상 상수를 동일 팔레트로 맞췄다.

## 법률 체크 반영

제품 설명에 다음 체크포인트를 넣었다.

- 채용: 채용서류 반환 안내, 개인정보 최소수집
- 근로계약: 임금, 소정근로시간, 휴일, 휴가 등 필수 근로조건 명시
- 유지관리: 근로시간, 휴일, 휴가 기록
- 교육관리: 직장 내 성희롱 예방교육 등 법정교육 이수 기록
- 퇴직관리: 퇴직 절차, 인수인계, 개인정보 보관 기준

> [!note]
> AEON은 법률 자문을 대신하지 않는다. 화면에서는 HR 담당자가 놓치기 쉬운 체크포인트를 먼저 보여주는 역할로 설명한다.

## 검증

- `npm run build` 성공
- `python -m py_compile backend\modules\applicants\competency_library.py backend\modules\applicants\router.py` 성공
- 핵심 신규/공통 UI 파일 ESLint 성공
- 전체 레거시 파일까지 포함한 broad lint는 기존 React hook/compiler 규칙 오류가 남아 있다.
