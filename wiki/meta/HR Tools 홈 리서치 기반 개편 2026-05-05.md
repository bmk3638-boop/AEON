# HR Tools 홈 리서치 기반 개편 2026-05-05

## 구현 요약

`frontend/src/pages/Home.jsx`를 Apple-style dark premium 홈으로 교체했다. 기존 파일은 한글 인코딩 손상과 JSX 손상이 심해 부분 수정보다 전체 교체가 안전했다.

전역 CSS `frontend/src/index.css`에는 AEON 3D intelligence model 애니메이션을 추가했다. 중심 코어, 3개의 회전 링, Mercer/IBM/Gallup 기능 카드가 천천히 떠 있는 구조다. `prefers-reduced-motion`을 지원해 모션 민감 사용자에게 애니메이션을 끈다.

## 반영한 컨설팅 리서치

- Mercer: 스킬 기반 인재관리. 채용, 학습, 승계, 성과, 보상까지 스킬을 공통 언어로 연결한다.
- IBM: AskHR와 watsonx Orchestrate 기반 HR agent. HR 문의 응답을 넘어 업무 분류, 라우팅, 실행 자동화까지 확장한다.
- Gallup: Q12 기반 몰입도 진단. 몰입을 측정하고 관리자 코칭 및 액션 플랜으로 연결한다.

## 홈에 추가된 제품 기능

1. `Skills Intelligence`
   - 직무별 스킬 매트릭스
   - 지원자-직무 적합도
   - 교육/승계 추천

2. `AskHR Agent`
   - 의도 분류
   - 업무 라우팅
   - 채용/근태/평가 액션 실행

3. `Engagement Pulse`
   - 몰입도 펄스
   - 팀별 리스크 보드
   - 관리자 액션 플랜

## 디자인 기준

`video-assets/hr-intro/DESIGN.md`를 기준으로 near-black 배경, Apple blue accent, 제한된 색상, 느리고 절제된 모션을 적용했다. 홈 hero는 기존 `public/intro.mp4`를 배경 영상으로 사용하고, 전면에는 3D intelligence model을 배치했다.

## 검증

- `npm ci`: 성공
- `npm run build`: 성공
- `npx eslint src/pages/Home.jsx`: 성공
- `npm run lint`: 실패. 원인은 이번 변경이 아니라 기존 여러 파일의 lint 문제다. 예: `AuthContext.jsx`, `ApplicantList.jsx`, `BioData.jsx`, `EvaluationForm.jsx`, `Recruiting.jsx`, `Signup.jsx`.
