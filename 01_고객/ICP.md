# 이상적 고객 프로필 (ICP)

> **Ideal Customer Profile** — 우리가 가장 잘 도울 수 있고, 가장 높은 가치를 느끼는 고객의 정의.
> 영업·마케팅·제안서 작성 시 이 기준으로 필터링하세요.

---

## 1차 타겟: 중소·중견기업 HR 실무 책임자

### 기업 조건

| 항목 | 조건 |
|------|------|
| 기업 규모 | 50~300인 |
| 산업군 | 제조업, IT서비스, 유통, 서비스업 |
| HR 팀 규모 | 1~5명 |
| 현재 단계 | HR 담당자가 있지만 시스템화 안 됨 |
| 예산 | AI/자동화에 연 500만~3,000만원 투자 가능 |
| 위치 | 수도권 (초기), 전국 (온라인 컨설팅) |

### 담당자 조건 (Decision Maker)

| 항목 | 내용 |
|------|------|
| 직책 | HR팀장 / HR Manager / 총무인사부장 |
| 나이대 | 30대 중반~40대 |
| 특성 | AI에 관심 있지만 기술 구현 능력 없음 |
| 고통 | 반복 업무 과부하, 채용 리드타임 길다, 온보딩이 체계 없다 |
| 목표 | 팀 효율화, 본인 업무 전략화, 경영진에게 인정받기 |

### Buy Signal (이런 고객이 우리에게 옵니다)

- "채용 공고 올리고 지원서 관리하는 데 주 X시간 쓰고 있다"
- "온보딩을 매번 수작업으로 한다"
- "ChatGPT 써봤는데 HR에 어떻게 쓸지 모르겠다"
- "자동화 도구 알아보다가 너무 복잡해서 포기했다"
- "HR 시스템 도입했는데 활용을 못 한다"

### Red Flag (이런 고객은 피하세요)

- 예산이 없지만 "나중에 잘 되면 주겠다"고 하는 곳
- IT 인프라가 너무 레거시해서 연동 불가
- 의사결정권자가 AI에 거부감이 심한 곳
- 요구 범위가 불명확하고 계속 바뀌는 곳 (스코프 크립 위험)

## 2차 타겟: 빠르게 성장하는 스타트업

### 기업 조건

| 항목 | 조건 |
|------|------|
| 기업 규모 | 30~100인 (Series A~B) |
| 특성 | 빠른 채용 필요, HR 프로세스 부재 |
| 담당자 | People Ops / HR 담당자 (1~2명) |
| 고통 | 사람은 빠르게 늘어나는데 HR 프로세스가 없다 |
| 예산 | 전략적 투자 가능 (VC 자금 있음) |

---

## 고객이 느끼는 Pain Point 우선순위

```
[채용 자동화]
1순위: 지원서 검토에 시간이 너무 많이 걸린다
2순위: 면접 일정 조율이 비효율적이다
3순위: 지원자 관리가 엑셀/이메일로만 된다

[온보딩 자동화]
1순위: 매번 새로 입사자마다 같은 걸 반복 설명한다
2순위: 온보딩 체크리스트가 없거나 지켜지지 않는다
3순위: 수습 기간 관리가 안 된다

[평가·성과]
1순위: 평가 시즌마다 집계에 시간을 너무 쓴다
2순위: 360도 평가 결과를 정리하기 어렵다

[기타]
1순위: AI 도구 어떻게 써야 할지 모르겠다
2순위: 자동화 도입했지만 아무도 안 쓴다
```

## 고객 여정 (Customer Journey)

```
인식 → 관심 → 고려 → 결정 → 온보딩 → 운영 → 재구매/추천
  ↑        ↑       ↑       ↑
콘텐츠   상담     제안서   계약
마케팅   미팅     발표     체결
```

### 각 단계별 우리의 역할
| 단계 | 고객 상태 | 우리가 할 일 |
|------|-----------|-------------|
| 인식 | "HR 자동화가 필요한 것 같다" | 콘텐츠, 사례 공유 |
| 관심 | "이런 회사가 있구나" | 무료 인사이트 제공 |
| 고려 | "우리한테 맞는지 알아보자" | Discovery 미팅 |
| 결정 | "이 회사에 맡기겠다" | 제안서, 견적 |
| 온보딩 | "계약했는데 잘 될까?" | 킥오프 미팅, 기대치 조율 |
| 운영 | "생각보다 좋다/나쁘다" | 정기 점검, 피드백 반영 |
| 추천 | "주변에 알려줘야겠다" | 레퍼런스, 케이스스터디 |

---

## 고객 규모별 접근 전략

| 규모 | 서비스 진입점 | 평균 계약 기간 |
|------|-------------|--------------|
| 50~100인 | Tier 1 진단 → Tier 2 1개 패키지 | 2~3개월 |
| 100~200인 | Tier 1 → Tier 2 복수 패키지 | 3~6개월 |
| 200~300인 | Tier 1 + 전체 설계 → 단계적 구축 | 6~12개월 |
