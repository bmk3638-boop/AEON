# 제안서 초안 생성

다음 정보를 기반으로 `$ARGUMENTS` 고객사를 위한 HR AI 자동화 컨설팅 제안서 초안을 작성합니다.

## 작업 순서
1. `CLAUDE.md` 읽기
2. `00_사업/서비스카탈로그.md` 읽기
3. `01_고객/ICP.md` 읽기
4. `01_고객/고객사/$ARGUMENTS.md` 가 있으면 읽기 (없으면 건너뜀)
5. 아래 형식으로 제안서 초안 작성 후 `03_영업/템플릿/제안서-$ARGUMENTS.md` 로 저장

## 제안서 형식

```markdown
# [고객사명] HR AI 자동화 컨설팅 제안서

**제안일**: YYYY-MM-DD
**제안사**: AEON
**버전**: v1.0

---

## 1. 현황 이해 및 과제
## 2. 제안 방향
## 3. 서비스 구성
## 4. 기대 효과 (ROI)
## 5. 추진 일정
## 6. 투자 비용
## 7. AEON 소개
```

톤: 전문적이고 신뢰감 있는 B2B 한국어. 결정권자(HR팀장·대표) 눈높이.
