# 미팅 준비 자료 생성

`$ARGUMENTS` 고객사와의 미팅을 준비합니다.

## 작업 순서
1. `01_고객/고객사/$ARGUMENTS.md` 읽기 (있으면)
2. `01_고객/ICP.md` 읽기
3. `00_사업/서비스카탈로그.md` 읽기
4. 미팅 준비 자료 생성

## 출력 형식

```markdown
# [고객사명] 미팅 준비 — YYYY-MM-DD

## 고객사 현황 요약
## 예상 Pain Point
## 준비할 질문 목록 (10개)
## 우리가 보여줄 것
## 예상 반론 및 대응
## 목표 (이 미팅에서 얻어야 할 것)
## 다음 단계 제안
```
