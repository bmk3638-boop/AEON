# 사업 개요

> 이 문서는 AEON의 사업 정체성과 방향을 정의합니다.
> 모든 의사결정의 기준이 되는 최상위 문서입니다.

---

## 비전 · 미션 · 가치

| 항목        | 내용                                                     |
| --------- | ------------------------------------------------------ |
| **비전**    | *(데이터와 자동화를 통해 HR의 의사결정을 표준화하고, 기업의 인사 경쟁력을 구조적으로 향상)* |
| **미션**    | 중소·중견기업의 HR 담당자가 반복 업무에서 벗어나 사람에 집중할 수 있도록             |
| **핵심 가치** | 완성도 / 신뢰 / 실행 가능성                                      |

---

## 우리가 다른 이유 (포지셔닝)

### 시장의 문제
- 대형 HR 솔루션사: 비싸고, 중소기업에 안 맞고, 구축만 하고 간다
- 프리랜서 자동화 개발자: 기술은 되지만 HR 도메인을 모른다
- HR 컨설팅사: 프로세스 설계는 하지만 실제 구현을 못 한다

### AEON의 위치
```
HR 도메인 지식 × AI/자동화 기술 × 운영 책임
= 중소기업이 실제로 쓸 수 있는 HR AI 자동화
```

## 사업 단계 로드맵

### Phase 1: 기반 구축 (현재)
- [ ] 서비스 정의 및 가격 책정
- [ ] 볼트(지식베이스) 구조화
- [ ] 첫 3개 고객사 확보 (레퍼런스용)
- [ ] 케이스스터디 제작

### Phase 2: 영업 체계화 (3~6개월)
- [ ] 콘텐츠 마케팅 시작 (LinkedIn, 뉴스레터)
- [ ] 파트너십 (HR 솔루션사, 노무사 등)
- [ ] 영업 파이프라인 CRM 구축

### Phase 3: 확장 (6~12개월)
- [ ] 팀 구성 (기술 파트너, 어드바이저)
- [ ] 자사 SaaS 또는 패키지 상품화
- [ ] 특정 산업군 버티컬 전문화
