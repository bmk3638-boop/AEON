## Tier 1 — HR 자동화 진단 (Discovery)

### 서비스 개요
고객사의 HR 프로세스 전체를 진단하고, AI·자동화 도입 우선순위와 로드맵을 제시하는 **입문형 서비스**.
본격 구축 전 필수 단계이며, Tier 2 진입의 기반이 됩니다.

### 포함 범위 (Deliverables)
- [ ] HR 프로세스 현황 인터뷰 (2~3회, 각 1.5시간)
- [ ] As-Is 프로세스 맵핑 문서
- [ ] 자동화 가능 영역 우선순위 매트릭스
- [ ] ROI 추정 시트 (시간 절감, 비용 절감)
- [ ] 90일 자동화 로드맵 제안서
- [ ] 도구 추천 리스트 (비용·난이도·효과 기준)

### 제외 범위
- 실제 구축·세팅 작업
- 외부 시스템 연동

### 소요 기간
2~3주

### 가격
→ `03_영업/가격표.md` 참조 *(작성 예정)*

## Tier 2 — 프로세스 자동화 구축

### 2-A. 채용 자동화 패키지

#### 자동화 대상 프로세스
1. 채용 공고 다채널 배포 자동화
2. 지원서 수집 → 1차 스크리닝 자동화 (AI 서류 검토)
3. 면접 일정 조율 자동화 (캘린더 연동)
4. 합격/불합격 통보 자동화
5. 레퍼런스 체크 자동화
6. 오퍼레터 발송 자동화

#### 기술 구성
| 기능 | 도구 |
|------|------|
| 워크플로우 | n8n 또는 Make |
| AI 스크리닝 | OpenAI API (GPT-4) |
| 일정 조율 | Google Calendar API |
| 데이터 저장 | Notion / Google Sheets |
| 커뮤니케이션 | Gmail API / Slack |

#### Deliverables
- [ ] 자동화 플로우 설계도
- [ ] n8n / Make 워크플로우 구축 및 테스트
- [ ] 관리자용 운영 가이드 (SOP)
- [ ] 2주 운영 동행 지원

#### 제외 범위
- ATS(지원자 추적 시스템) 자체 개발
- 기존 ATS와의 복잡한 커스텀 연동

---

### 2-B. 온보딩 자동화 패키지

#### 자동화 대상 프로세스
1. 입사 전 (Pre-boarding): 서류 수집, 장비 신청, 환영 메시지
2. 입사 당일: 계정 생성, 오리엔테이션 자료 자동 발송
3. 1주차~1개월: 체크인 알림, 과제 배정, 진행률 트래킹
4. 수습 평가: 설문 자동 발송, 결과 집계

#### 기술 구성
| 기능 | 도구 |
|------|------|
| 워크플로우 | n8n 또는 Make |
| 문서 자동화 | Google Docs API / Notion |
| 커뮤니케이션 | Slack, Gmail |
| 체크리스트 | Notion / Asana |

#### Deliverables
- [ ] 온보딩 마스터 플로우 설계
- [ ] 단계별 자동화 구축 및 테스트
- [ ] 온보딩 자료 템플릿 세트
- [ ] 관리자·매니저용 운영 가이드

---

### 2-C. 성과·평가 자동화 패키지

#### 자동화 대상 프로세스
1. 평가 주기 알림 자동 발송
2. 자기평가 · 동료평가 설문 배포
3. 결과 집계 및 시각화
4. 면담 일정 자동 조율
5. 평가 기록 아카이브

#### 기술 구성
| 기능 | 도구 |
|------|------|
| 설문 | Google Forms / Typeform |
| 집계·분석 | Google Sheets + Apps Script / AI |
| 일정 | Google Calendar |
| 저장 | Notion |

---

### 2-D. 퇴직 프로세스 자동화

#### 자동화 대상 프로세스
1. 퇴직 신청 접수 → 관련 부서 알림
2. 인수인계 체크리스트 자동 생성
3. 계정·장비 회수 프로세스
4. 퇴직 인터뷰 설문 자동 발송
5. 퇴직 서류 자동 생성 (재직증명서 등)

## Tier 3 — AI 도구 도입 컨설팅

### 3-A. HR 도구 선정 · 세팅

#### 서비스 내용
- 고객사 규모·예산·기술 수준에 맞는 도구 선정
- 선정 도구 초기 세팅 및 계정 구성
- 기존 데이터 마이그레이션 지원

#### 주요 검토 도구
| 카테고리 | 도구 옵션 |
|----------|-----------|
| HR 관리 | Notion HR, 그리팅, Jobda |
| 채용 | 원티드, 사람인 ATS, Lever |
| 자동화 | n8n, Make, Zapier |
| AI 보조 | ChatGPT, Claude, Perplexity |
| 커뮤니케이션 | Slack, Teams |

---

### 3-B. 팀 교육 프로그램

#### 교육 모듈 (선택 조합 가능)
| 모듈 | 대상 | 시간 |
|------|------|------|
| AI 기초 + HR 활용 오리엔테이션 | HR팀 전체 | 2시간 |
| ChatGPT/Claude 실무 활용 | HR담당자 | 3시간 |
| n8n 자동화 기초 | 운영 담당자 | 4시간 |
| 데이터 기반 HR 의사결정 | HR팀장·임원 | 2시간 |

## Tier 4 — 유지보수 리테이너

### 서비스 내용
구축 완료 후 안정적 운영을 위한 **월정액 지속 지원** 서비스.

#### 포함 내용
- 월 X회 화상 점검 미팅
- 자동화 오류 대응 (영업일 기준 24시간 내)
- 소규모 수정·개선 작업 (월 X시간 포함)
- 분기별 프로세스 최적화 리뷰
- 신규 AI 도구 동향 공유

#### 등급 (작성 예정)
- Basic / Standard / Premium

## 서비스 조합 추천 패턴

| 고객 상황 | 추천 조합 |
|-----------|-----------|
| "AI 도입하고 싶은데 뭐부터 해야 할지 모르겠다" | Tier 1 → Tier 2-A or B |
| "채용이 너무 힘들다" | Tier 1 (채용 집중) → Tier 2-A |
| "온보딩이 중구난방이다" | Tier 2-B |
| "도구는 있는데 안 쓴다" | Tier 3-B (교육) |
| "다 해줬으면 좋겠다" | Tier 1 → 2 전체 → 4 (리테이너) |
