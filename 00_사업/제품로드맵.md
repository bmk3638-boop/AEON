# 제품 로드맵

> Top-Down 원칙: 궁극 목표를 먼저 정의하고, 역산으로 단계를 설계한다.

---

## 궁극 목표 (+1)

### Vision: 벡터DB 기반 경영 종합 대시보드 SaaS

```
인사 → 재무 → 회계 → 구매 → 생산관리 → 품질관리
                    ↓
          벡터DB 기반 데이터 통합
                    ↓
       고위 경영자 의사결정 종합 대시보드
                    ↓
            SaaS 형태로 전 기업 공급
```

- 모든 부서 데이터를 SQL + 벡터DB로 통합
- C-Level이 자연어로 질의 → 즉시 인사이트
- 로컬 AI 기반 (데이터 외부 유출 없음)
- 법률 기반 시스템 위에 구축 (노동법, 세법, 개인정보보호법)

## 4대 핵심 아이디어 (단계별 진입점)

### 아이디어 1. 채용플랫폼 자동화 시스템
**[ 현재 연구 중 → `[[02_솔루션/HR자동화모듈/01-채용자동화-연구]]` ]**

#### 기능 구성
```
채용플랫폼 (사람인, 잡코리아 등)
        ↓
[요건 필터링] 자격조건 충족 이력서만 자동 선별
        ↓
[자동 다운로드] 클라우드 or 로컬 저장
        ↓
[자동 대시보드화] 지원자 현황 시각화
        ↓
[회사별 분리] 하위구조: 사내 부서별 분류
```

#### 세부 기능
- 요건 충족 필터 (직무, 경력, 자격증, 학력 등)
- 이력서 자동 파싱 (이름, 연락처, 스킬 구조화)
- 회사별 / 부서별 / 공고별 폴더 자동 생성
- 대시보드: 지원자 수, 합격률, 단계별 현황

#### 기술 스택 (연구 중)
- 사람인/잡코리아 공식 기업 API (최우선 검토)
- Playwright 브라우저 자동화 (API 불가 시)
- Python 이력서 파싱 (PDF → 구조화 데이터)
- Notion / Google Sheets 대시보드

#### 법률 검토 필수
- 개인정보보호법: 지원자 동의 기반 수집만 허용
- 채용플랫폼 이용약관 준수 필수
- 상세: `[[05_운영/SOP/법률검토-채용데이터]]` *(작성 예정)*

### 아이디어 2. 회사별 HR 챗봇
**[ 설계 단계 ]**

#### 목적
- 과거 지원자 데이터 검색
- 키워드 기반 인재 검색 ("Python 3년차 경력자 있어?")
- 인재풀 유지 및 재활용

#### 기능 구성
```
누적된 이력서 데이터 (아이디어 1의 결과물)
        ↓
벡터DB 인덱싱 (ChromaDB, Pinecone 등)
        ↓
HR 챗봇 인터페이스 (Slack / 웹앱)
        ↓
자연어 검색 → 후보자 추천
```

#### 기술 스택 (잠정)
- 벡터DB: ChromaDB (로컬) or Pinecone (클라우드)
- LLM: Claude API or 로컬 Ollama
- 인터페이스: Slack 봇 or 웹 대시보드
- 보안: 회사별 완전 격리 (멀티테넌시)

### 아이디어 3. 근태관리 자동화 시스템
**[ 설계 단계 ]**

#### 두 가지 방식
| 방식 | 연동 방법 | 특징 |
|------|-----------|------|
| **SECOM 연동** | SECOM 근태 API or 엑셀 내역 파싱 | 기존 시스템 활용 |
| **WiFi 위치기반** | 사내 WiFi SSID 감지 → 출퇴근 기록 | 별도 장치 불필요 |

#### 기능 구성
```
출퇴근 데이터 (사원번호 기반)
        ↓
자동 근태 집계 (지각/조퇴/결근/초과근무)
        ↓
급여 계산 연동 (아이디어 4)
        ↓
근태 현황 대시보드
```

#### 법률 검토 필수
- 근로기준법: 근로시간 기록 의무
- 위치정보보호법: WiFi 기반 위치 수집 시 동의 필요

### 아이디어 4. 급여 자동화 ERP
**[ 장기 설계 단계 ]**

#### 기능 구성
```
근태 데이터 (아이디어 3)
        ↓
급여 항목 자동 계산
  - 기본급, 수당, 식대, 교통비
  - 초과근무수당 (근로기준법 기반 자동 계산)
  - 4대보험 공제 자동화
  - 세금 계산 (갑근세, 지방소득세)
        ↓
급여명세서 자동 생성 + 발송
        ↓
회계 시스템 연동 (아이디어 +1)
```

#### 특이사항 처리 자동화
- 중도 입퇴사 일할 계산
- 육아휴직 / 산재 / 병가
- 성과급, 인센티브 지급 규칙

```

        ↑↓
[3. 근태 시스템] ↔ [4. 급여 ERP]

[2. HR 챗봇] (전체 데이터 검색)

[+1. 종합 대시보드] (C-Level 의사결정)
```
