# MERCER / GALLUP / IBM 컨설팅 방향성 벤치마킹

> AEON HR 자동화 컨설팅 방향성 설계 참고용. 세 글로벌 HR 컨설팅 기업의 핵심 방법론 정리.

---

## 1. MERCER — 스킬 기반 인재관리 + AI 통합

### 핵심 프레임워크
- **Global Talent Trends**: 매년 12,000+ 비즈니스/HR 리더 설문 → 트렌드 도출 (2026년까지 지속)
- **Skills-Based Talent Management**: 스킬을 채용·육성·성과·승계 전반에 통합 (74개국 1,100명 HR리더 데이터 기반)
- **IPE (International Position Evaluation)**: Know-How · Problem Solving · Accountability 3요소로 직무 가치 정량화

### 2025~2026 컨설팅 방향
| 영역 | 방향성 |
|------|--------|
| AI 코칭 | HR 리더의 25%가 AI 코칭으로 관리자·개인 역량 강화 |
| 성과관리 | AI가 성과 트렌드 포착 → 관리자에게 넛지 → 개인화 개입 제안 |
| 승계관리 | 평가 도구로 핵심/시니어 직급 승계 슬레이트 강화 |
| 리더십 | 리더십 개발 + 승계 전략 + 기간세대 협업(intergenerational teaming) 결합 |
| 디지털 전환 | "Digital by Design" — 인간-기계 팀워크 내재화 |

### AEON 벤치마킹 포인트
- [ ] 스킬 갭 분석 리포트 자동 생성 (채용 → 평가 → 육성 연동)
- [ ] IPE 기반 직무 가치 평가 모듈 (평가관리에 적용)
- [ ] AI 코칭 넛지 — 성과 이상 시 관리자에게 알림 자동화

## 2. GALLUP — 강점 기반 몰입도 + Q12 진단

### 핵심 프레임워크
- **Q12 서베이**: 12개 문항으로 직원 몰입도 과학적 측정 (전 세계 표준)
- **CliftonStrengths 34 themes → 4 Domains**:
  - Executing / Influencing / Relationship Building / Strategic Thinking
- **Boss to Coach**: 관리자를 코치로 전환 — 강점 발견·활용·개발 지원

### 강점 집중의 효과 (Gallup 연구)
| 접근법 | 몰입:비몰입 비율 |
|--------|----------------|
| 강점 중심 관리 | **60:1** |
| 약점 중심 관리 | 2:1 |

→ 강점 집중이 몰입도 30배 차이 만들어냄

### 컨설팅 방식
1. Q12 진단 → 조직/팀/개인 몰입도 수준 파악
2. CliftonStrengths 평가 → 개인 강점 프로파일 도출
3. 강점-몰입 연계 코칭 프로그램 설계
4. 관리자 대상 "Boss to Coach" 전환 교육
5. 지속 측정 → 개선 루프

### AEON 벤치마킹 포인트
- [ ] Q12 기반 몰입도 자가진단 모듈 (평가관리 내 탑재)
- [ ] CliftonStrengths 4 Domains → 직원 강점 카드 자동 생성
- [ ] 관리자 코칭 리포트 — 팀원 강점 기반 업무 배치 추천

## 3. IBM — AI 기반 HR 전환 + 인재 인텔리전스

### 핵심 프레임워크
- **HR & Talent Transformation**: Design Thinking + Agile + DevOps + Generative AI 결합
- **Value Activation Roadmap**: HR을 비용 센터 → 고부가가치 비즈니스 파트너로 전환
- **watsonx HR Agents**: AI 에이전트가 HR 프로세스 전반 자동화

### 4대 서비스 영역
| 영역 | 내용 |
|------|------|
| 채용 | 글로벌 RPO, 후보자 경험 최적화, AI 스크리닝 |
| 스킬 | 역량 평가 + AI 학습 추천 + 디지털 아카데미 |
| 경력 | Blue Match — 직원 스킬↔직무 매칭 경력 경로 시각화 |
| 측정 | 성과 아웃컴 정량화, 스킬 갭 추적, ROI 측정 |

### IBM HR 전환 3대 목표 (2010년 이후 지속 발전)
1. 프로세스 단순화로 운영 효율화
2. 데이터 기반 인재 의사결정
3. 관리자에게 올바른 도구 제공

### AEON 벤치마킹 포인트
- [ ] HR을 "비즈니스 파트너"로 포지셔닝하는 컨설팅 언어 채택
- [ ] Blue Match형 경력 경로 시각화 (개발관리 모듈)
- [ ] 스킬 갭 → 학습 추천 → 성과 측정 루프 자동화
- [ ] watsonx 방식의 HR 에이전트 → AEON 챗봇에 확장
