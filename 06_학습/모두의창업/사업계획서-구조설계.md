# 모두의 창업 — 사업계획서 구조 설계

> **전략 원칙**: 심사위원은 10분 안에 판단한다.
> → 처음 3장에서 승부를 결정짓는다.
> → 모든 주장은 숫자(BI)로 뒷받침한다.

---

## 비유 → 벤치마킹 → 실전응용

### 비유
> 사업계획서는 **빙산**이다.
> 심사위원에게 보이는 것은 수면 위 10%.
> 나머지 90%는 Obsidian 지식베이스 안에 있다.
> 보이지 않는 90%가 탄탄해야 보이는 10%가 흔들리지 않는다.

### 벤치마킹 사례
| 회사 | 초기 포지셔닝 | 실제 비전 | 교훈 |
|------|--------------|-----------|------|
| **Workday** | 중소기업 HR SaaS | 글로벌 엔터프라이즈 ERP | 작게 시작, 크게 확장 |
| **Notion** | 메모 앱 | 올인원 팀 OS | 도구가 철학이 됨 |
| **Salesforce** | 중소기업 CRM | AI+CRM 생태계 | 플랫폼화 전략 |
| **Kakao** | 메신저 | 생활 플랫폼 | 핵심 채널 선점 후 확장 |

> 공통점: 모두 "작고 명확한 문제 해결"로 시작해서 플랫폼이 되었다.

### 실전응용 (AEON)
```
지금 심사위원에게 보여줄 것:
"HR 담당자의 반복업무 X시간을 Y%로 줄인다"
→ 명확하고 측정 가능한 가치

숨겨둘 것 (묻는다면 공개):
글로벌 SaaS, ERP, 1조 매출
→ 지금 제시하면 신뢰도 하락
```

### 보통 회사 vs 컨설팅 회사

| 구분 | 보통 회사의 사업계획서 | AEON (컨설팅)의 사업계획서 |
|------|----------------------|--------------------------|
| 문제 정의 | 추상적 ("HR이 비효율적이다") | 수치화 ("채용 1건당 평균 23일, X만원 소요") |
| 해결책 | 기능 나열 | ROI 계산 포함한 프로세스 설계 |
| 시장 분석 | 전체 시장 규모만 | 접근 가능 시장(SAM) + 초기 타겟 명확화 |
| 팀 | 인원 소개 | 각 역할의 AI 에이전트 포함 설명 |
| 수익 모델 | 1가지 | 프로젝트+리테이너 복합 + SaaS 로드맵 |

## 사업계획서 목차 설계

### 표지 + 요약 (Executive Summary)
- 1장: 우리가 해결하는 문제 (Pain)
- 핵심 지표 3개만: 시간 절감 XX%, 비용 절감 XX만원, 고객 만족도 XX

### 본문 구성

#### 1장. 문제 정의 (Problem)
- 파일: `[[06_학습/모두의창업/01-문제정의]]` *(작성 예정)*
- 핵심: HR 담당자의 실제 데이터 (설문, 인터뷰)
- BI: 반복업무 시간 비율, 채용 리드타임, 온보딩 비용

#### 2장. 해결책 (Solution)
- 파일: `[[06_학습/모두의창업/02-해결책]]` *(작성 예정)*
- 핵심: 우리 서비스의 Before/After
- BI: ROI 계산, 자동화 전환율

#### 3장. 시장 분석 (Market)
- 파일: `[[06_학습/모두의창업/03-시장분석]]` *(작성 예정)*
- TAM: 한국 HR 테크 전체 시장
- SAM: 50~300인 중소기업 HR 자동화 시장
- SOM: 우리가 1년 안에 실제로 잡을 수 있는 시장

#### 4장. 비즈니스 모델 (Business Model)
- 파일: `[[06_학습/모두의창업/04-비즈니스모델]]` *(작성 예정)*
- 수익 흐름: 프로젝트 → 리테이너 → SaaS
- 단가, 계약 기간, LTV

#### 5장. 경쟁 분석 (Competition)
- 파일: `[[06_학습/모두의창업/05-경쟁분석]]` *(작성 예정)*
- 직접 경쟁: HR 컨설팅사, 자동화 에이전시
- 간접 경쟁: SaaS HR 툴 (그리팅, 잡다 등)
- 차별화: HR 도메인 × AI/자동화 × 운영책임

#### 6장. 실행 계획 (Execution)
- 파일: `[[06_학습/모두의창업/06-실행계획]]` *(작성 예정)*
- 3개월 / 6개월 / 12개월 마일스톤
- 리소스 배분

#### 7장. 팀 (Team)
- 파일: `[[06_학습/모두의창업/07-팀구성]]` *(작성 예정)*
- 창업자 역량
- AI 에이전트 역할 구조 설명

#### 8장. 재무 계획 (Financials)
- 파일: `[[07_재무/재무계획-모두의창업]]` *(작성 예정)*
- 3년 손익 예측
- 손익분기점 (BEP)
- 자금 사용 계획

## 작성 우선순위

```
지금 당장 작성:
1. 01-문제정의 (데이터가 핵심 — 가장 설득력 있는 파트)
2. 03-시장분석 (숫자로 설득)
3. 07-재무계획 (BEP 계산)

다음 단계:
4. 02-해결책
5. 04-비즈니스모델
6. 05-경쟁분석
7. 06-실행계획
8. 07-팀구성
```

## 심사 통과 체크리스트

- [ ] 모든 주장에 출처 있는 숫자 첨부
- [ ] 한 장당 핵심 메시지 1개
- [ ] Before/After 명확 (자동화 전/후 비교)
- [ ] ROI 계산 포함 (투자 대비 절감 효과)
- [ ] 경쟁 우위 3줄 이내로 요약
- [ ] 창업자가 왜 이 사업인지 (Fit) 명확
- [ ] 12개월 실행 가능한 계획 (과하지 않게)
- [ ] 재무 보수적 시나리오 포함
