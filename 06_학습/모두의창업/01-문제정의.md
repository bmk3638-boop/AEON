# 1장. 문제 정의 (Problem)

> **전략**: 심사위원이 "맞아, 이거 진짜 문제다"라고 고개를 끄덕이게 만든다.
> 감이 아닌 숫자로, 우리 경험이 아닌 시장 데이터로.

---

## 비유 → 벤치마킹 → 실전응용

### 비유
> HR 담당자는 **소방관이 된 건축가**다.
> 원래는 조직을 설계하고 사람을 키워야 하는데,
> 매일 불 끄는 반복업무(채용공고 올리기, 면접 일정 조율, 입사서류 수집)에 하루를 다 쓴다.
> AEON은 소방 자동화 시스템을 설치해서, 건축가가 건축을 다시 할 수 있게 한다.

### 벤치마킹 사례
| 회사/사례 | 문제 정의 방식 | 결과 |
|----------|--------------|------|
| **Workday** | "HR 담당자가 스프레드시트에서 60% 시간을 낭비" → 수치화로 투자 유치 | IPO 성공 |
| **Greenhouse** | "채용 리드타임 평균 42일, 1인당 채용비용 $4,200" → 문제를 돈으로 환산 | 유니콘 |
| **국내 그리팅** | "채용 공고 관리 수작업 → 주 15시간 낭비" 데이터 제시 | 시리즈B |

### 실전응용 (AEON)
```
우리의 문제 정의 공식:
"국내 50~300인 기업 HR 담당자는 전체 업무의 [X]%를 반복 업무에 소비한다.
이를 금액으로 환산하면 연간 [Y]만원의 인력 비용 낭비다.
우리는 이것을 [Z]%까지 줄인다."

→ X, Y, Z를 채우는 것이 지금 당장 해야 할 숙제
```

### 보통 회사 vs 컨설팅 회사

| | 보통 회사 | AEON (컨설팅) |
|--|----------|--------------|
| 문제 서술 | "HR이 힘들다" | "채용 1건에 평균 23일, 담당자 월 40시간 소요" |
| 근거 | 경험담 | 설문 데이터 + 업계 리포트 인용 |
| 범위 | 우리 경험 | 시장 전반의 구조적 문제 |
| 공감 유도 | 감성적 호소 | 숫자로 심사위원 스스로 계산하게 함 |

## 핵심 문제 3가지

### 문제 1. 채용 프로세스의 비효율
```
현실:
- 채용공고 1건을 여러 플랫폼에 올리는 시간: 평균 2~3시간
- 지원서 1차 검토에 소요 시간: 지원자 1명당 평균 7분
- 100명 지원 시 → 약 700분(11.7시간) 소요
- 면접 일정 조율 (이메일/전화): 평균 2~4일 소요

AEON 자동화 후:
- 공고 배포: 10분 (자동 멀티채널 배포)
- AI 1차 스크리닝: 100명 → 3~5분 이내
- 일정 조율: 캘린더 연동으로 당일 확정
```

**[데이터 수집 필요]**: 국내 HR 담당자 설문 또는 업계 리포트 인용

### 문제 2. 온보딩 프로세스의 부재
```
현실:
- 신규 입사자 온보딩을 매번 수작업으로 진행
- 담당자 1명이 신규 입사자 1명 온보딩에 소요하는 시간: 평균 8~16시간
- 온보딩 체크리스트 없는 기업 비율: [데이터 필요]

AEON 자동화 후:
- 입사 전 서류 수집: 자동화 (담당자 개입 0)
- 오리엔테이션 자료 발송: 자동
- 1주차 체크리스트: 자동 배포 + 진행률 실시간 추적
```

### 문제 3. AI 도입 의지 있지만 방법 모름
```
현실:
- ChatGPT 사용 경험 있는 HR 담당자 비율: [데이터 필요]
- "AI를 HR에 어떻게 써야 할지 모른다"는 응답: [데이터 필요]
- AI 도구 도입 시도 후 포기 비율: [데이터 필요]

AEON 가치:
- 도구 선정 → 세팅 → 교육 → 운영 전 과정 책임
```

## 수치화 목표 (데이터 수집 체크리스트)

수집해야 할 데이터:

- [ ] 국내 50~300인 기업 HR 담당자 1인당 평균 업무 시간 분포
- [ ] 채용 1건당 평균 소요 시간 (국내 통계)
- [ ] 채용 1건당 평균 비용 (국내 통계)
- [ ] HR 담당자 중 "반복업무가 50% 이상"이라 응답한 비율
- [ ] AI/자동화 도입 의향 있지만 미실행 비율
- [ ] 국내 HR 테크 시장 규모 및 성장률

참고 출처 (탐색 필요):
- 고용노동부 HR 관련 통계
- 한국HR협회 리포트
- 잡코리아/사람인 채용 트렌드 리포트
- Gartner HR Technology 리포트 (글로벌)

## 문제 정의 최종 문장 (초안)

> "국내 중소기업 HR 담당자는 전체 업무 시간의 절반 이상을
> 채용공고 관리, 면접 일정 조율, 입사서류 수집 등 **AI로 즉시 자동화 가능한 반복업무**에 소비하고 있다.
> 이를 금액으로 환산하면 50~300인 기업 기준 연간 평균 **[Y]만원**의 인력 낭비이며,
> HR 담당자의 **전략적 역할(채용 브랜딩, 조직문화, 인재 육성)이 실종**되는 구조적 문제를 낳는다."

→ [Y] 값 계산 필요: HR 담당자 연봉 × 반복업무 비율 × 기업당 HR 인원 수
