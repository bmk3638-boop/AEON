### VoltAgent

- **GitHub**: [github.com/VoltAgent/voltagent](https://github.com/VoltAgent/voltagent)
- **분류**: TypeScript 기반 AI 에이전트 엔지니어링 플랫폼
- **핵심 기능**: 멀티에이전트 워크플로우, Human-in-the-loop, 메모리 지속성, VoltOps 모니터링
- **사용 타이밍**: 컨설팅 납품물 → SaaS 제품 전환 시점
- **활용 시나리오**: 고객사가 자체적으로 HR 자동화를 운영하는 플랫폼 구축

### notebooklm-py

- **GitHub**: [github.com/teng-lin/notebooklm-py](https://github.com/teng-lin/notebooklm-py)
- **분류**: Google NotebookLM 비공식 Python API
- **핵심 기능**: 이력서/문서 자동 업로드 → 오디오 요약, 마인드맵, 퀴즈, 슬라이드 생성
- **사용 타이밍**: 고객사 납품물 차별화가 필요한 시점 (내부 실험 먼저)
- **활용 시나리오**: 이력서 분석 결과를 오디오 브리핑 + 마인드맵으로 변환해서 납품
- **주의**: 비공식 라이브러리 — Google API 변경 시 망가질 수 있음

### PM2

- **GitHub**: [github.com/Unitech/pm2](https://github.com/Unitech/pm2)
- **분류**: Node.js 프로덕션 프로세스 매니저
- **핵심 기능**: 무중단 재시작, 클러스터 로드밸런싱, 실시간 모니터링, 자동 장애 복구
- **사용 타이밍**: VoltAgent 기반 SaaS 제품 서버 운영 시작 시점
- **활용 시나리오**: 에이전트 서버 여러 개를 안정적으로 24/7 운영


### Bun

- **GitHub**: [github.com/oven-sh/bun](https://github.com/oven-sh/bun)
- **분류**: All-in-one JavaScript 런타임 (Node.js 대체)
- **핵심 기능**: Node.js보다 3~4배 빠른 실행 속도, 내장 번들러·테스트러너·패키지매니저
- **사용 타이밍**: VoltAgent SaaS 전환 시 서버 성능 최적화 필요 시점
- **활용 시나리오**: 에이전트 API 서버를 Node.js에서 Bun으로 교체 시 응답속도 대폭 향상

### Sourcegraph

- **GitHub**: [github.com/sourcegraph/sourcegraph](https://github.com/sourcegraph/sourcegraph)
- **분류**: 코드 인텔리전스 플랫폼 (코드 검색·탐색·분석)
- **핵심 기능**: 대규모 코드베이스 전체 검색, AI 코드 어시스턴트(Cody), 코드 인사이트 대시보드
- **사용 타이밍**: 개발팀이 5인 이상으로 커지고 코드베이스가 복잡해지는 시점
- **활용 시나리오**: 여러 에이전트 모듈 간 의존성 파악, 코드 리뷰 자동화
