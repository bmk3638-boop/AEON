# 실수 보완 누적 (Lessons Ledger)

> 작업 중 반복되는 실수와 보완책을 한 곳에 누적한다. 대표님 지시(2026-06-24): "과거에서 지속적으로 실수를 보완". 매 세션 새 교훈을 위에 추가하고, 세션로그에서 `[[실수_보완_누적]]`로 링크한다.
> 형식: **증상 → 원인 → 보완(다음엔 이렇게)**. 같은 실수가 또 나오면 횟수를 표시(재발 N회).

## 검증·증거 (가장 자주 어기는 영역)

- **코드만 읽고 '양호/완료' 단정 (재발 주의)**
  - 증상: grep/코드 리딩만으로 "설계자모드 양호"라 보고 → 실제 화면은 엉망이었음(2026-06).
  - 원인: 컴파일된다 ≠ 화면이 멀쩡하다. 레이아웃 겹침·런타임 에러는 빌드로 안 잡힘.
  - 보완: 'broken'은 증거(수치·빌드)로 단정 가능하나 '양호/완료'는 **렌더·실행 확인 없이 단정 금지**. 이 PC는 **WDAC 정책으로 브라우저 바이너리 실행 차단(스크린샷 불가)** → 대안: 레이아웃 수치 계산 + `npm run build` + API 스모크 + 대표님 직접 확인.

- **거짓 모지바케 함정 (2026-06-24)**
  - 증상: `curl | python -m json.tool`로 API 응답 보니 한글이 `\udcec…`로 깨져 인코딩 버그로 오인.
  - 원인: Windows python stdin이 cp949로 UTF-8 바이트를 잘못 디코딩한 **파이프 아티팩트**(실제 응답은 정상).
  - 보완: 인코딩 의심 시 `curl -o file` 후 **hexdump(`xxd`)나 Read 도구로 원본 바이트 확인**. 한글 결과는 파일로 떠서 Read. 터미널 출력엔 `PYTHONIOENCODING=utf-8`.

- **경로 스타일 함정 (2026-06-24)**
  - 증상: `/c/Users/...`(MSYS) 경로를 Windows python/curl에 넘겨 FileNotFound.
  - 보완: 네이티브 도구엔 **Windows 경로(`C:/...`)** 로 통일.

- **curl 한글 본문 깨짐 (2026-06-24)**
  - 증상: `curl -d '{"reason":"한글"}'`가 400 `error parsing the body` → override 코드 버그로 오인.
  - 원인: Windows bash가 curl 인자의 한글을 깨뜨려 JSON 바이트 손상(cp949 함정 친척). ASCII 본문은 정상 200.
  - 보완: **한글 JSON 본문은 파일로 만들어 `--data-binary @file.json` 전송**하거나 ASCII로 1차 검증. 400 'parsing body'면 코드보다 본문 인코딩부터 의심.

## 변경의 부수효과

- **정직화/판정 로직 변경 시 다운스트림 점검 (2026-06-24)**
  - 증상: QA Gate를 정직화(release_ready를 실제 검사 결과로) 하자, 그 값에 의존하던 **테넌트 기능 enable이 전부 막힘**(데모가 실검사 실패 → release_ready False).
  - 보완: 게이트/판정값을 바꿀 땐 **그 값을 읽는 모든 다운스트림 플로우**(여기선 tenant 활성·release-candidate)를 같이 점검하고 영향을 미리 고지. 필요 시 사유기록 override를 함께 설계.

## 도구·운영

- **도구 호출 형식 오류 (재발)**: 도구 호출을 잘못된 태그로 작성해 파싱 실패. → 정확한 호출 형식 엄수.
- **세션 종료 루틴**: 종료 시 git push + 옵시디언 저장까지 반드시 완료. 미커밋 있으면 새 작업 전 커밋 정리 먼저.
- **데모 직전 pytest 금지**: 통합테스트는 실제 데모 DB를 오염시킴(`itest-`/`cert-` 등). 유닛만(`-m "not integration"`)은 무DB라 안전. 통합은 conftest cleanup이 도는 session-close에서.
- **작업 운영**: 분석→보고→승인→구현→기록. MVP 스코프 임의 확장 금지. 챕터별 완결(구현+검증+보고) 후 다음 챕터(2026-06-24 CLAUDE.md 절대원칙화).
