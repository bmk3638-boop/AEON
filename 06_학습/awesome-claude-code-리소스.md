# Awesome Claude Code 리소스

> Claude Code 생태계의 도구, 프레임워크, 스킬 큐레이션 모음

## AEON 관점에서 핵심 포인트

### 1. Claude API 연동으로 사람인 파이프라인 고도화
- 현재: 키워드 단순 텍스트 매칭 → 점수 산출
- 업그레이드 방향: Claude가 이력서 전문을 직접 읽고 적합도 평가 + 요약 생성
- 납품물 품질이 크게 올라감 (컨설팅 차별화 포인트)

### 2. 멀티에이전트 오케스트레이션
- claude-squad, claude-flow 등 병렬 처리 프레임워크
- 여러 공고를 동시에 처리하는 구조로 확장 가능
- HR 자동화 납품 규모 확장 시 필요

### 3. 스킬 시스템 활용
- 반복 작업을 슬래시 명령어로 자동화
- 이력서 분석, 공고 매칭, 보고서 생성 등을 스킬화 가능

### 4. 사용량 모니터링 도구
- 토큰 사용량 추적 및 비용 분석 대시보드
- 고객사 납품 시 비용 산정 근거로 활용 가능
