## 전체 자동화 플로우

```
1. 품질관리 담당자 채용 페이지 이동
   → https://hiring.saramin.co.kr/applicant-manage/recruit/53551943

2. 배주환 체크박스 클릭
   → input[aria-label="배주환 선택"]

3. 선택 툴바에서 "자료 저장" 버튼 클릭
   → body.innerText에 "자료 저장" 나타나면 버튼 클릭

4. 1단계 모달: "모든 지원 서류" 선택
   → button.innerText.includes("모든 지원 서류")

5. 2단계 모달 설정:
   → 저장 항목: radio[value="all"] (한 폴더 모든 이력서)
   → 저장 사유: [role="combobox"] → "전형 관리" 클릭
   → 비밀번호: ActionChains + send_keys + keyup/input 이벤트

6. "저장" 버튼 클릭 → ZIP 파일 다운로드
```

## 핵심 기술 해결 포인트

### 문제 1: "자료 저장" vs "자료저장"
- **증상**: 띄어쓰기 없는 "자료저장"으로 XPath 탐색 실패
- **해결**: `innerText.includes('자료 저장')` 방식으로 교체

### 문제 2: 모달 외부 combobox 선택
- **증상**: `document.querySelector('[role="combobox"]')`가 내비게이션의 공고 필터를 선택
- **해결**: `[role="dialog"]` 내부에서만 탐색

### 문제 3: React 비밀번호 입력 안 됨
- **증상**: `nativeInputValueSetter`로 값 세팅해도 저장 버튼 클릭 시 모달이 안 닫힘
- **해결**: ActionChains + `send_keys` + `keyup`/`input` 이벤트 조합
```python
ActionChains(driver).click(f)
    .key_down(Keys.CONTROL).send_keys("a").key_up(Keys.CONTROL)
    .send_keys(Keys.DELETE).perform()
f.send_keys(PW)
el.dispatchEvent(new KeyboardEvent("keyup", {bubbles:true}))
el.dispatchEvent(new InputEvent("input", {bubbles:true}))
```

### 문제 4: 다운로드 경로 설정
- CDP `Page.setDownloadBehavior`로 바탕화면 폴더에 저장
