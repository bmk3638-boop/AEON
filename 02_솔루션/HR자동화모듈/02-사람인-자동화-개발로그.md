---

## 아키텍처 요약

```
Chrome (9222포트) ─► pipeline.py
                        ├─ A: 스크랩 지원자 감지 (파란별 버튼)
                        ├─ B~H: ZIP 다운로드 (체크→자료저장→서류→폴더→사유→PW→저장)
                        ├─ I: ZIP 압축 해제 (cp437→UTF-8 인코딩 처리)
                        ├─ J: PDF 텍스트 추출 (pdfplumber)
                        ├─ K: 키워드 스코어링 (8개 카테고리, ~80개 키워드)
                        └─ L: HTML 대시보드 생성
                              → Obsidian 로그 (성공로그/)
                              → Windows 알림
```

---

## 실행 방법

```bash
# 기본 (config.json의 활성 공고 처리)
python pipeline.py

# 전체 공고 스캔
python pipeline.py --all

# 키워드 검색
python pipeline.py --공고 품질관리

# 강제 재로그인
python pipeline.py --재로그인

# Windows 작업 스케줄러 등록 (매일 09:00)
python pipeline.py --스케줄

# 완료 후 대시보드 자동 열기
python pipeline.py --대시보드
```

또는 바탕화면 배치파일 → `menu.ps1` 메뉴 실행

## v3 추가 기능 (2026-04-26)

| 기능 | 설명 |
|------|------|
| `config.json` | 공고 목록 외부화. 코드 수정 없이 공고 추가/삭제 |
| `state.json` | 오늘 이미 다운로드한 인원 조합 추적 → 재다운로드 방지 |
| retry 로직 | ZIP 다운로드 최대 3회 재시도 |
| Windows 알림 | 완료 시 시스템 트레이 풍선 알림 |
| `--스케줄` | Windows 작업 스케줄러 자동 등록 |
| `--대시보드` | 완료 후 브라우저 자동 열기 |
| `logs/` | 날짜별 로그 파일 (`logs/YYYY-MM-DD.log`) |

공고 추가: `recruits` 배열에 항목 추가. `active: false`로 일시 비활성화 가능.
