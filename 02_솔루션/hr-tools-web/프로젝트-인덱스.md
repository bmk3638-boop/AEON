## 색 시스템 (60:30:10)

```css
/* 테마 변수 — frontend/src/index.css */
--color-primary:       #0F8A5F   /* 10% : CTA 초록 (버튼, 강조) */
--color-secondary:     #334155   /* 30% : 구조색 slate (헤더, 아이콘) */
--color-bg:            #F3F8F4   /* 60% : 바탕 */
--color-thead:         #f1f5f9   /* 테이블 헤더 배경 (중성 회색) */
--color-primary-soft:  #E8F4EC  /* 초록 soft (선택 행, 키워드 뱃지) */
--color-border:        #D8E7DE
--color-muted:         #61736A
```

> 규칙: **thead는 항상 `C.thead`(중성 회색), 초록(`C.soft`)은 선택/강조 상태에만**

## 공유 유틸리티 (pages 폴더)

| 파일 | 역할 |
|------|------|
| `acqData.js` | `C` (색 상수), `STATUS`, `MOCK` 데이터 전체 |
| `acqStore.js` | Zustand 전역 상태 (interviews, applicants, finalPassed 등) |
| `acqLayout.jsx` | `PageHeader`, `Panel`, `Metric`, `WorkflowNav`, `PrimaryButton`, `SecondaryButton` |
| `acqNavigation.js` | `ACQ_RECRUITING_SECTIONS` — 모집관리 사이드바 항목 정의 |
| `acqScoring.js` | 부서별 선발기준 점수 계산 |
| `acqSynapse.js` | HR Synapse (AI 분석) 연동 |
| `DataCommandBar.jsx` | 검색/필터/정렬/CSV/카운터 공통 바 (`noBorder` prop 지원) |
