# AEON HR Design System

## Theme Direction

AEON HR should support three design directions:

1. Green Tone
   - Calm, trustworthy, HR SaaS-like.
   - Suitable for default admin pages.

2. Korean Brush-Touch Premium
   - Uses subtle Korean visual identity.
   - Suitable for branding, landing, or premium mode.

3. Apple / Wanted-like Modern Minimal
   - Clean, airy, premium, modern.
   - Recommended for home and major dashboards.

**Current default is Green Tone.** Do not switch themes automatically.

## Green Tone Palette (Current Default)

```
Primary CTA:   #0F8A5F
Deep nav/text: #173B2F
Background:    #F3F8F4
Soft surface:  #E8F4EC
Border:        #D8E7DE
```

Orange (`#FF5E00`, `#FF7A59`, `#FFF4EF`), blue, or black-dominant palettes
must not be used without explicit user approval.

## Layout Principles

- Clear sidebar navigation.
- Stable navbar.
- Optional right panel for contextual insights.
- Card-based layouts.
- Avoid dense enterprise clutter.
- Prefer modern spacing, clean hierarchy, and readable typography.

## UX Principles

- Show important information first.
- Reduce repeated input.
- Make HR workflows visible.
- Use dashboards for status and decisions, not decoration.
